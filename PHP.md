+ Uma Linha

```PHP

```

+ Várias Linhas

```PHP

```

## Variáveis

Em **PHP** uma variável não é declarada mas sim atribuida. Para chamar uma variável em **PHP** usa-se o **$** seguido do nome da variável.

```PHP
$variavel = "valorVariavel";
```
